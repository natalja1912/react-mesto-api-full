# Название проекта
**Сервис Mesto**
Сервис Mesto: интерактивное приложение с регистрацией пользователей, куда пользователи могут добавлять фотографии, удалять их и ставить лайки.

## О чем проект
 * Фотографии интересных мест в России и не только
 * Карачаевск
 * Эльбрус
 * Домбай


## Технологии
 * HTML, CSS
 * адаптивная верстка
 * React JS
 * NODE JS
